## Overview
This is a flask web app which predicts the first innings score of Indian Premier League(IPL) with the help of Rgeressor model. The dataset is taken from https://github.com/codophobia/CricketScorePredictor credits to Shivam Mitra. It contains the score made on each ball of the matches played in IPL from 2008 to 2017.
